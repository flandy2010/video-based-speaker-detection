## 项目介绍
本项目是基于视频的说话人识别。在给定一段视频的情况下，判定这段视频中的说话人是谁。目前只能够判断单一说话人的视频。如果需要对于多人轮流说话的视频判断每段时间的说话人，可以通过其他方式判定可能的说话人切换点，随后对每小段视频进行判断。

## 原理说明
本项目主要依据人脸关键点的变动。人脸关键点序号如图：
